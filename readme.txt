Проект был сделан в Visual studio 2019 Console application на языке C#
1. Program.cs - Содержит основной метод Main и будет использовать другие классы для выполнения различных операций.
2. FileOperations.cs - Этот файл будет содержать методы для работы с файлами.
3. PerformanceTesting.cs - Тестирование производительности
4. MathOperations.cs - математические операции (Факториал,Фибоначчи)
5. ConcurrentProcessing.cs - параллельная обработка с ConcurrentBag
